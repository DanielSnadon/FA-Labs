### -q ###

Обычные:

./main -q 0.1 1 2 -3

#
 Случай a = 1.000000, b = 2.000000, c = -3.000000. 
Ответ: два решения - 1.000000 и -3.000000. 

 Случай a = 2.000000, b = -3.000000, c = 1.000000. 
Ответ: два решения - 1.000000 и 0.500000. 

 Случай a = -3.000000, b = 1.000000, c = 2.000000. 
Ответ: два решения - -0.666667 и 1.000000. 

 Случай a = 1.000000, b = -3.000000, c = 2.000000. 
Ответ: два решения - 2.000000 и 1.000000. 

 Случай a = -3.000000, b = 2.000000, c = 1.000000. 
Ответ: два решения - -0.333333 и 1.000000. 

 Случай a = 2.000000, b = 1.000000, c = -3.000000. 
Ответ: два решения - 1.000000 и -1.500000. 
#

Необычные:

./main -q 0.0000001 2 0 3

#
 Случай a = 2.000000, b = 0.000000, c = 3.000000. 
Ответ: дискриминант отрицательный, у уравнения нет действительных корней. 

 Случай a = 0.000000, b = 3.000000, c = 2.000000. 
Ответ: уравнение не является квадратным. Решение единственно и равно -0.666667. 

 Случай a = 3.000000, b = 2.000000, c = 0.000000. 
Ответ: два решения - 0.000000 и -0.666667. 

 Случай a = 2.000000, b = 3.000000, c = 0.000000. 
Ответ: два решения - 0.000000 и -1.500000. 

 Случай a = 3.000000, b = 0.000000, c = 2.000000. 
Ответ: дискриминант отрицательный, у уравнения нет действительных корней. 

 Случай a = 0.000000, b = 2.000000, c = 3.000000. 
Ответ: уравнение не является квадратным. Решение единственно и равно -1.500000. 
#

Ошибочные:

./main -q 0.1 1 2 - # Ошибка ввода: неверный ввод значения третьего коэффициента. 
./main -q -1 0 0 0 # Ошибка ввода: неверный ввод значения эпсилон. 

./main -q 0.1 1 2 -5 5 # Ошибка ввода: флаг -q требует 4 числа для работы. 

### -m ###

Обычные:

./main -m -1 1 # Первое число кратно второму. 

Необычные:

./main -m 2147483647 1 # Первое число кратно второму. 

Ошибочные:

./main -m 0 5 # Ошибка ввода чисел: для флага необходимы ненулевые числа. 
./main -m 2147483648 1 Ошибка ввода числа: число переполняет ячейку памяти.
./main -m 5.5 1 # Ошибка ввода числа: недопустимый символ в числе. 

### -t ###

Обычные:

./main -t 0.001 3 4 5 # Числа могут являться длинами сторон прямоугольного треугольника 
./main -t 0.001 1 1 1 # Числа не могут являться длинами сторон прямоугольного треугольника 

Необычные:

./main -t 0.6 3 4 5.05 # Числа могут являться длинами сторон прямоугольного треугольника 
./main -t 0.01 3 4 5.05 # Числа не могут являться длинами сторон прямоугольного треугольника 

Ошибочные:

./main -t 0.1 -1 2 3 # Длины сторон треугольника должны быть положительными числами. 
./main -t 0.01 1e100 1e200 1.414e100 # Ошибка функции: переполнение ячейки памяти вследствие предельных вычислений.
